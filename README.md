1. Click **"Load Sample Data"** in the sidebar
2. Start chatting: "What are my top spending categories?"
3. Explore the Dashboard and Transactions tabs

Built with:
- Groq (FREE LLM)
- LangChain
- Plaid
- Streamlit
- Python

### To run:
python -m venv venv
source venv/bin/activate 
pip install -r requirements.txt
streamlit run app.py
